🛒 Shopping Cart Application

A modern and responsive Shopping Cart application built using React.js, designed to simulate a real-world e-commerce experience with reusable components, scalable architecture, and dynamic cart management functionality.

🚀 Features
✅ Product listing page
✅ Add to cart functionality
✅ Remove items from cart
✅ Increase / decrease product quantity
✅ Dynamic cart total calculation
✅ Responsive user interface
✅ Reusable React components
✅ State management using React Hooks
✅ Routing and navigation support
✅ Clean and scalable folder structure
🛠️ Tech Stack
Frontend
- React.js
- JavaScript (ES6+)
- HTML5
- Tailwind CSS
- React Router DOM
- Development Tools
- Visual Studio Code
- Git & GitHub
- npm

⚙️ Installation & Setup
Clone the repository
git clone https://github.com/sagar-vaishnav/shopping-cart.git
Navigate to the project directory
cd shopping-cart
Install dependencies
npm install
Run the application
npm start
Open in browser
http://localhost:3000
